Transfer-Out Risk Predictive Model

Pension transfers represent one of the biggest threats to AUM retention — yet most providers only react after a customer has already decided to leave. I built an ML model that detects early signals of transfer-out risk, giving Smart Pension's product and outreach teams a window to intervene proactively. The model won 1st place at the company-wide hackathon and is now deployed in production, directly protecting assets under management.

Move AI — AI-Powered Home Moving Assistant

6.7 million people move house in the UK every year, facing 100+ admin records to update — utility providers, councils, banks, GP surgeries. Move AI eliminates this burden with personalised onboarding, an AI assistant, voice agent for elderly users (55+), document scanner, smart checklist with 100+ tasks, local area guide, and a marketplace.

Voice agent powered by ElevenLabs + Simli WebRTC real-time lip-sync avatar. Document scanner uses Claude Vision API to extract account numbers and addresses from uploaded PDFs and photos, auto-filling the checklist. Accessibility modes: plain English, step-by-step guidance, large text, multilingual support.

Stack: Anthropic Claude (chat + vision), ElevenLabs, Simli WebRTC, Supabase, Resend, React + TypeScript + Node.js, Railway
